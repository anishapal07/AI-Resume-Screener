#  AI-Powered Resume Ranker

A high-performance text analysis tool designed to bridge the gap between job requirements and candidate profiles. By utilizing advanced **Natural Language Processing (NLP)** techniques, this tool provides an objective, mathematical score for candidate relevance.

##  Core Technology
The system transforms raw text into intelligence using:
* **TF-IDF Vectorization:** To identify and weight key industry terms (e.g., Python, Java, DSA).
* **Cosine Similarity:** To measure the mathematical "distance" between a CV and a Job Description.



##  Key Features
* **Automated Screening:** Eliminates manual reading of hundreds of CVs.
* **Keyword Extraction:** Automatically identifies critical skills in resumes.
* **Instant Scoring:** Provides a percentage match to help recruiters prioritize high-potential candidates.

##  Installation & Usage
1. **Clone the repository:**
   `git clone https://github.com/anishapal07/AI-Resume-Screener.git`
2. **Install dependencies:**
   `pip install scikit-learn`
3. **Execute:**
   Run `python main.py` to see the AI scoring in action.
##  What it does:
* **Text Extraction:** Reads data from resumes.
* **AI Analysis:** Uses **NLP** (Natural Language Processing) to find keywords.
* **Scoring:** Gives a match percentage based on the job description.

##  Tech Stack:
