# AI Resume Screener 

This is a **Computer Science Engineering** project created for my internship. It uses Artificial Intelligence to help recruiters find the best candidates.

##  What it does:
* **Text Extraction:** Reads data from resumes.
* **AI Analysis:** Uses **NLP** (Natural Language Processing) to find keywords.
* **Scoring:** Gives a match percentage based on the job description.

##  Tech Stack:
* **Language:** Python
* **Libraries:** Scikit-Learn (TF-IDF, Cosine Similarity)
* **Tools:** VS Code, Git

##  How to run:
1. Clone the repository.
2. Add your details to `testresume.txt`.
3. Run `python main.py`.
